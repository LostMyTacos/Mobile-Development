This is a Repo thats main purpose is to show my mobile-development skills.
All Projects are built using C# and Xamarin/Xamarin.Forms, and were only tested via an Android Emulator.

MyMovieApp:

	This Application uses a SQLite Database and a Web API: TMDB(A Movie API). It allows users to search,
	for movies and add them to a movie watch list.

SwipeQuiz: 

	This quiz utilizes a swipe feature to generate interaction from the user and test their knowledge 
	on slightly silly topics.

TipCalculator:
	
	This is a generic TipCalculator built in Xamarin.Forms.
